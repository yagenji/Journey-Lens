# Journey Lens — セットアップと運用ガイド

世界各地の写真とタイムラプスを、管理画面から随時追加できるサイトです。トップは「表紙に出す」写真のモザイク、地図や写真から国ごとの特集ページ（体験エッセイ＋写真・タイムラプス）へ入ります。

## ファイル構成
```
.
├── index.html        … サイト本体（content.json を読み込む）
├── content.json      … 国・写真・タイムラプス・エッセイのデータ（管理画面が編集）
├── admin/            … 管理画面（/admin）と項目定義 config.yml
├── uploads/          … アップした画像・短い動画
└── README.md
```
写真・映像・エッセイは現在すべてサンプルです。エッセイはご自身の体験に書き換えてください。

## ローカル確認
`content.json` の読み込みには簡易サーバーが必要です（無い場合もサンプルは表示されます）。
```
python3 -m http.server 8000   →  http://localhost:8000
```

## 公開（Cloudflare Pages 例）
1. GitHub に一式をアップロード。
2. Cloudflare Pages でリポジトリ接続（ビルドなし／出力ディレクトリ `/`）。
3. `https://〇〇.pages.dev` が発行。Netlify でも同様。

## 管理画面（/admin）
Sveltia CMS（GitHub 連携）。旧 Netlify Identity は廃止のため、最新の公式手順で認証を設定してください。
- 手順: https://github.com/sveltia/sveltia-cms
- `admin/config.yml` の `repo:` と `base_url:` を自分のものに置換。
- 公開URL末尾に `/admin` でログイン → 国・写真・タイムラプス・エッセイを編集。

## 構成と各項目
- **トップのモザイク**：写真の「表紙に出す」をONにしたものだけが並びます（各国から厳選）。クリックでその国ページへ。1枚もONが無いときは各国の代表写真を自動表示。
- **地図**：国の緯度・経度でピンと旅の軌跡を自動描画。ピンや一覧から国ページへ。
- **国ページ（特集）**：扉写真＋国名＋導入文＋体験エッセイ（段落は空行区切り、1段落目の後に引用が大きく入る）＋写真・タイムラプスの組み。
- **国の追加**：新しい国を足して緯度・経度を入れると、地図・モザイク・特集ページが自動で増えます。

## タイムラプス（短尺ループ／30秒以上の長尺）
- クリック再生（feature）… 長尺向け。サムネ＋再生ボタン → 拡大再生。
- 自動ループ（loop）… 短い無音クリップ向け。ギャラリー内で静かにループ。
- 長尺・大容量は **動画URL**（YouTube / Vimeo / MP4直リンク）に貼るとサイトもリポジトリも軽いまま。短い無音ループのみ動画ファイル直アップに対応。

## 固定の文言
サイト名「Journey Lens」、タグライン、自己紹介、連絡先などは `index.html` 内のテキストを編集してください。
